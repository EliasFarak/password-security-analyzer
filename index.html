04 — ABOUT

How It Works

01

Entropy Analysis

Calculates Shannon entropy using charset size and password length. Formula: E = L × log₂(N) where L = length, N = charset size.

02

Pattern Detection

Identifies common weak patterns: numeric sequences, keyboard walks, dictionary words, repeated characters, and predictable substitutions.

03

Brute-Force Estimation

Estimates crack time assuming 10⁹ guesses/second (modern GPU). Calculates N^L total combinations and divides by attack rate.

04

Dictionary Check

Checks against a curated list of the most commonly used passwords worldwide. Flags matches as critical vulnerabilities.
